# 艺术极简个人主页 - 设计系统 (Stitch)

## 1. 核心理念与创意导向: "流动美术馆" (The Living Gallery)
该设计系统建立在“流动美术馆”这一创意坐标之上。它拒绝了传统网页模板那种静态、方正的特性，转而追求一种流动的、社评式的体验，让内容能够“呼吸”。我们不是在构建一个网站，而是在策划一场数字展览。

通过“高对比度极简主义”实现高级感。将深邃如真空的背景色 (#0e0e0e) 与充满活力、如电光般的品牌色 (#f3ffcd) 配对，营造出一种聚焦的生命力感。布局必须是有意图的非对称设计，利用广阔的留白让内容仿佛悬浮在画廊空间中。

## 2. 颜色与表面哲学
色调中心在于深邃的影调深度与单一的高饱和“生命”点缀色。

### “无边框”原则
设计师禁止使用 1px 的实线边框来进行区域划分或包含。边界应通过以下方式创建：
- **背景切换**：从 `surface` (#0e0e0e) 切换到 `surface_container_low` (#131313)。
- **色调过渡**：在 `surface_container` 的不同层级间使用柔和渐变来定义边缘。

### 表面层级与嵌套
将 UI 视为“黑曜石玻璃”的物理图层。使用 `surface-container` 令牌创建突出或凹陷的感觉：
- **基础层**：`surface` (#0e0e0e) 用于主视口。
- **下沉元素**：`surface_container_low` (#131313) 用于次要内容区域。
- **抬升元素**：`surface_container_high` (#20201f) 用于卡片或交互模块。

### “玻璃与渐变”规则
为了唤起“生命力”，`primary` (#f3ffcd) 和 `secondary` (#e9e958) 不仅仅是平铺填充，更是光源。
- **毛玻璃效果 (Glassmorphism)**：浮动导航使用 `surface_container` 60% 透明度，辅以 `24px` 的背景模糊。
- **活力渐变**：在极具冲击力的 CTA 按钮上，应用从 `primary` 到 `primary_container` (#c5fe00) 的线性渐变，模拟发光效果。

## 3. 字体设计: 艺术社评感
我们将极具个性的衬线体与功能性的无衬线体配对，架起“艺术性”与“实用性”之间的桥梁。

- **展示与标题 (Newsreader)**：这款衬线体是我们的“艺术”支柱。在 Hero 区域使用 `display-lg` (3.5rem) 并加大字间距，营造出高端艺术学刊标题页的质感。
- **正文与 UI (Manrope)**：这款无衬线体是我们的“实用”支柱。它提供了干净、现代的锚点。始终在长文本中使用 `body-lg` (1rem) 以保持高端且易读的质感。
- **刻意的比例差**：直接从 `display-lg` 跳跃到 `body-lg` 来创造戏剧性。避免使用“中庸”的大小，以保持层级的锐利与前卫。

## 4. 高度与深度
该系统中的深度是有机的、氛围感的，而非结构化的。

- **图层原理**：避免在标准卡片上使用投影。相反，将 `surface_container_highest` (#262626) 卡片置于 `surface_dim` (#0e0e0e) 背景之上。14% 的亮度差异足以产生“柔和提升”感。
- **环境阴影**：对于真正“悬浮”的元素（如浮动导航），使用模糊半径为 `40px`、透明度为 `6%` 的阴影，并带上 `primary` (#f3ffcd) 的色调，营造微光感而非黑暗空洞。
- **“幽灵边框”回退**：若无障碍要求必须有描边，使用 `outline_variant` (#484847) 并设置 **15% 透明度**。它应当是被“感觉到”而非被“看到”。

## 6. 核心组件规范

### 浮动导航栏 (Floating Navigation Bar)
- **样式**: `surface_container` (70% opacity), `backdrop-blur: 20px`。
- **形状**: `full` (9999px) 圆角。
- **交互**: 滚动时，容器宽度应略微缩小，透明度增加至 90%。

### 艺术卡片 (Artistic Cards)
- **结构**: 无边框。使用 `surface_container_high` (#20201f)。
- **悬停效果**: 缩放 `1.02`，文本颜色从 `on_surface_variant` 过渡到 `primary`。

### 活力按钮 (Vitality Buttons)
- **主要按钮**: 背景 `primary` (#f3ffcd)，文字 `on_primary` (#4f6700)。圆角 `md` (1.5rem)。
